Home Automation System (Senior Design Project)

Me and my team developed a Home Automation System that would control the unlocking of your door, light dimming, and temperature control.

Developed the hardware using 3 Particle Core microcontrollers, along with schematics and modules for the specific components.

Architected and developed the software for the microcontrollers using the Arduino Wireframe language.

Created a real-time web application to communicate to the hardware wirelessly using JavaScript/jQuery and HTML/Bootstrap.

Submitted approved technical documents and presented to a panel of industry professionals.

Microgrid Simulation Model (Volunatry)

Develop simulation program for an Electrical Engineering Masters Thesis.

Using Python, created and simulated complex mathematical equations taking input from user and data sheets

Architected efficient code and algorithm to implement load shedding, which will take certain parameters and then determine which power sources to be turned off and then turned back on in the correct order. This is the main innovation in the thesis.

Ran extensive unit testing, as the complexity of this project makes testing components together very difficult.

Created documentation for the student and advisor to have a full understanding of the code and how it operates, so it is easy to build on for future endeavors.
